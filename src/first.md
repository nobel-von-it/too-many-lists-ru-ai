# Плохой односвязный стек (A Bad Singly-Linked Stack)

Эта глава будет *безусловно* самой длинной, так как нам нужно познакомить вас практически
со всем Rust, и мы собираемся создавать некоторые вещи «трудным путем», чтобы лучше
понять язык.

Мы поместим наш первый список в `src/first.rs`. Нам нужно сказать Rust, что `first.rs` — это
то, что использует наша библиотека. Все, что для этого требуется, — поместить это в начало
`src/lib.rs` (который Cargo создал для нас):

```rust ,ignore
// в lib.rs
pub mod first;
```
